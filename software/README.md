# CYTHS - Software solution

## Dependencies

The following main components must be installed on the Operating System in order to have solution working:

 | Component | Version   |
 |:----------|:----------|
 | Bash      | >= 4.3.30 |
 | PHP       | >= 8.0    |
 
### PHP module

The following PHP module is needed:

 | Module              | Version |
 |:--------------------|:--------|
 | php_curl            | >= 8.0  |
