### Components

Components used are:

| Component | Reference           |
|:----------|:--------------------|
| R1        | 2.2 KΩ 5%           |
| R2        | 27 Ω 5%             |
| LED       | Yellow LED          |
| IC1       | ATtiny85-20PU       |
| RF1       | 433 MHz transmitter |
| RF2       | 433 MHz receiver    |
