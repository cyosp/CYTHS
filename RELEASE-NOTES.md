##### 5.0.1 (2023-07-17)

 * /!\ Crontab condition breaking change /!\

   - Comparison operators move from: gt, ge, ... to standard ones: >, >=, ...
   - Restrict JSON path evaluation to left condition side
 * Dependency changes:
   * Remove NodeJS
   * PHP >= 8.0
 * Log received code as decimal value
